To check the Data Layer json data

    • Hit the following url http://host:port/ITag2/getDataLayer?
    • Pass the parameters after "?" like pagetype=submit&solution=cardapplication&applyflowid=123
    • Ex: https://xyz.com/getDataLayer? pagetype=submit&solution=cardapplication&applyflowid=123
    • hit the url

To update the Data Layer Json data

    • Hit the following url http://host:port/ITag2/updateDataLayer?
    • Pass the parameters after "?" like pagetype=submit&solution=cardapplication&applyflowid=123
    • Pass the modified Data Layer json in body
    • hit the url
